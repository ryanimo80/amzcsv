# amzcsv

Tool amzcsv: 
- Mục đích: Tool sử dụng để tự động hóa công việc tạo file csv inventory để import vào các marketplace như Amazon, Ebay, Wish...
- Chức năng:
	+ Quản lý file PNG design
	+ Quản lý danh sách từ khóa theo niche
	+ Quản lý profile mockup (giá cả, màu sắc, loại sản phẩm)
	+ Xuất mockup cho từng loại sản phẩm
	+ Xuất CSV inventory cho các marketplace
